Invert Binary Tree

Easy LeetCode 226

Given the root of a binary tree, invert the tree, and return its root.

solution.js 
const invertTree = (root) => {
  if (!root) return null;
  
  // Swap children
  const temp = root.left;
  root.left = root.right;
  root.right = temp;
  
  // Recursively invert
  invertTree(root.left);
  invertTree(root.right);
  
  return root;
};
Complexity Analysis
  • Time Complexity: O(n)
  • Space Complexity: O(h)
